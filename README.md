# Langchain-Message-History
This repository contains practice code while learning LangChain message history, ChatGroq models, prompt templates, and maintaining conversational state using LangChain utilities.

## Contents
- ChatGroq model usage
- Message History with RunnableWithMessageHistory
- ChatMessageHistory (in-memory storage)
- Prompt templates and MessagesPlaceholder
- Maintaining session-based chat history
- Passing session_id through config

## Purpose
This repo is only for learning and practicing LangChain message history concepts step-by-step.

## How to run
Create a virtual environment, install dependencies from `requirements.txt`, and open the notebook to run the practice code.
